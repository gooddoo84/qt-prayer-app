## 1. 배경 및 목적

### 1.1 7분법 기도란?
7가지 단계로 나누어 체계적으로 기도하는 방법:
1. **묵상** — 성경 말씀을 읽고 묵상
2. **찬양과 경배** — 하나님을 찬양하며 경배
3. **감사** — 감사한 것들을 고백
4. **회개** — 죄를 고백하고 회개
5. **간구** — 개인적인 필요를 기도
6. **중보기도** — 다른 사람을 위한 기도
7. **특별대상을 위한 기도** — 특정 대상/상황을 위한 기도

### 1.2 목표
- 매일 성경 1장을 순서대로 읽기 (마태복음 2장부터 시작 → 성경 전체)
- 쉬운성경 버전으로 말씀 표시
- AI(GPT)가 해당 장을 쉽게 요약
- 7단계 기도를 단계별 페이지로 안내
- 각 단계에서 메모 작성 가능
- 완료 시 매일 다른 격려 메시지 표시

## 2. 요구사항

### 2.1 핵심 기능 (Must Have)

| ID | 기능 | 설명 |
|----|------|------|
| F-01 | 오늘의 성경 말씀 | 매일 1장씩 순서대로 표시 (쉬운성경 버전, 마태복음 2장부터) |
| F-02 | AI 성경 요약 | GPT가 해당 장의 핵심 내용을 쉽게 요약 |
| F-03 | 7단계 기도 UI | 각 단계별 개별 페이지, 순서대로 진행 |
| F-04 | 단계별 메모 | 각 기도 단계에서 메모 입력 및 저장 (localStorage) |
| F-05 | 완료 격려 메시지 | 7단계 완료 시 매일 다른 격려/축복 메시지 |
| F-06 | 진행 추적 | 현재 몇 번째 장인지, 어디까지 읽었는지 추적 |

## 3. 기술 스택

### 3.1 아키텍처

```
[사용자 브라우저]
  ├── 정적 웹앱 (HTML/CSS/JS)
  │   ├── 7단계 기도 UI (SPA)
  │   ├── 메모 저장 (localStorage)
  │   └── 진행 추적 (localStorage)
  │
  ├── 성경 API ──→ [외부 성경 API 서버]
  │                  (쉬운성경 버전)
  │
  └── AI 요약 ──→ [OpenAI API / AI Gateway]
                    (GPT로 성경 요약)
```

### 3.2 기술 선택

| 레이어 | 기술 | 이유 |
|--------|------|------|
| **프레임워크** | Next.js 16 (정적 export) | GitHub Pages 배포, SEO |
| **스타일** | Tailwind CSS | 빠른 개발, 다크모드 |
| **성경 API** | bible-api.com 또는 getbible.net | 무료, 한글 성경 지원 |
| **AI 요약** | OpenAI GPT API | 성경 요약 품질 우수 |
| **데이터 저장** | localStorage | 로그인 없이 로컬 저장 |
| **배포** | GitHub Pages | 무료 정적 호스팅 |
| **아이콘** | Lucide Icons | 경량, 기도 관련 아이콘 |

### 3.3 성경 읽기 순서

마태복음 2장부터 시작하여 성경 전체를 순서대로:
```
신약: 마태복음 2장 → ... → 요한계시록 22장
구약: 창세기 1장 → ... → 말라기 4장
총 1,189장 (매일 1장 = 약 3년 3개월)
```
